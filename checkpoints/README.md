# Required Checkpoint Files

The following files are required in this directory for the Fashion Segmentation service. They are not included in the repository due to their size.

## Fashionpedia R50-FPN

Location: `checkpoints/fashionpedia-r50-fpn/`

- `checkpoint`
- `model.ckpt.data-00000-of-00001` (~371 MB)
- `model.ckpt.index`
- `model.ckpt.meta` (~13.6 MB)

Archive: `fashionpedia-r50-fpn.tar.gz` (~345 MB)

Please ensure the `fashionpedia-r50-fpn` directory is populated with these files or extract the archive into this directory.
